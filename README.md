# mapLocation

批量转换地址为经纬度 [mapLocation](http://maplocation.sjfkai.com)

# 介绍

一个 `web` 小工具，提供地名批量转换为经纬度，并下载的功能。

基于 [React](https://reactjs.org/) 和 [Ant Design](https://ant.design/)

托管在 [github](http://github.com) 和 [coding](http://coding.net/)（为了百度SEO） 上。
